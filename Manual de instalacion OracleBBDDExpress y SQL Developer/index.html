Introducción

Esta es una guía paso a paso, diseñada para aquellos que no tienen experiencia previa en la instalación de bases de datos. Aquí explicaremos de manera sencilla cómo instalar Oracle Database Express Edition, una versión gratuita y ligera del sistema de gestión de bases de datos Oracle. Además, también veremos cómo instalar Oracle SQL Developer, una herramienta gráfica que facilita la administración y consulta de la base de datos.

Oracle Database Express permite almacenar y gestionar datos sin necesidad de invertir en licencias costosas. SQL Developer , por otro lado, proporciona una interfaz gráfica que permite interactuar con la base de datos sin necesidad de escribir comandos complejos.

Con esta guía, tendrás todo listo para comenzar a trabajar con Oracle Database Express Edition en cuestión de minutos.

Oracle Database Express

Descarga del Software

Para descargar Oracle Database Express Edition, acceda al siguiente enlace: Oracle Database XE

Seleccione la primera opción si utiliza el sistema operativo Windows.


El archivo descargado será un paquete comprimido en formato .zip.

Instalacion del Software

  1. Extraccion de archivos: Una vez completada la descarga, descomprima el archivo .zip que contiene la distribución de Oracle Database Express Edition.

  2. Ejecución del instalador:Acceda a la carpeta extraída y ejecute el archivo setup.exe con permisos de administrador.

  3. Proceso de instalación:

    • En la primera ventana del asistente de instalación, haga clic en "Siguiente".

    • Lea y acepte los términos y condiciones del software.

    • Se le dará la opción de cambiar la ubicación de la instalación. Se recomienda dejar la ruta predeterminada, pero puede modificarla según sus necesidades.

    • Establezca una contraseña para los usuarios SYS, SYSTEM y PDBADMIN. En esta guía utilizaremos la contraseña "123456" (no se recomienda en entornos productivos).

    • Revise el resumen de las rutas de instalación y haga clic en "Instalar".


Configuraciones Adicionales

Durante el proceso de instalación, se le solicitará habilitar permisos de red a través del firewall del sistema. Asegúrese de permitir el acceso cuando se le solicite.

Finalizacion de la instalación

Una vez concluida la instalación, haga clic en "Finalizar" y ejecute Oracle Database Express Edition para verificar su correcto funcionamiento.




Para iniciar sesión, ingrese el usuario y la contraseña establecidos durante la instalación. Si la autenticación es exitosa, el sistema estará operativo.


Instalación de SQL Developer

Descarga del Software

Para la administración de la base de datos, descargue e instale Oracle SQL Developer desde el siguiente enlace: Oracle SQL Developer

  1. Descargue la versión que incluye el kit de desarrollo de Java (JDK) 17 si no lo tiene previamente instalado.




  2. Extraiga el contenido del archivo .zip y mueva la carpeta a una ubicación conveniente, como la carpeta "Documentos".


  3. (Opcional) Cree un acceso directo del ejecutable para facilitar su acceso. Se recomienda enviarlo al escritorio.



  4. Abra SQL Developer por primera vez:

    • Si aparece una ventana preguntando si desea exportar preferencias de una instalación anterior, seleccione "No" si parte de una configuración nueva.



    • Cuando se le consulte si desea permitir la monitorización del uso del software, desmarque la casilla y "aceptar".



Con esto, la instalación y configuración de Oracle Database Express Edition y SQL Developer estará completa y lista para su uso.
